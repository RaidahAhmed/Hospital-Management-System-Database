## Hospital-Management-System-Database
- Name: Raidah Ahmed
- Project: SQL Database System Assignment

## Project Description
This project is a Hospital Management System database developed using SQL Server.

The database is designed to manage hospital operations such as:
- Patient records
- Doctor and nurse management
- Appointments
- Admissions
- Treatments
- Prescriptions
- Billing
- Room allocation

The system uses relational database concepts including:
- Primary Keys
- Foreign Keys
- Constraints
- INNER JOINS
- Aggregate Functions
- VIEW implementation

## Database Tables Created
The following tables were created:
1. Department
2. Doctor
3. Nurse
4. Patient
5. Appointment
6. Admission
7. Room
8. Treatment
9. Prescription
10. Bill
11. Nurse_Assignment

## Advanced Database Feature
A VIEW named:
`Patient_Appointment_View`
was implemented to simplify retrieval of appointment information by combining data from multiple related tables.

## How to Run the Project
### Step 1
Open SQL Server Management Studio (SSMS).

### Step 2
Create a new database then run the provided SQL script (`hospital_management_system.sql`)

### Step 3
Execute the SQL script to:
- Create tables
- Insert records
- Create relationships
- Create the VIEW

### Step 4
Run sample queries provided in the script.
